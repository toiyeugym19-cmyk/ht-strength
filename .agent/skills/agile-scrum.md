# Agile & Scrum

## Scrum Roles

- **Product Owner**: Manages backlog, priorities
- **Scrum Master**: Facilitates process
- **Development Team**: Build the product

## Scrum Events

### Sprint Planning
- Duration: 2 weeks (typical)
- Plan: Select user stories from backlog
- Commit: What can be delivered

### Daily Standup (15 min)
- What did I do yesterday?
- What will I do today?
- Any blockers?

### Sprint Review
- Demo completed work
- Gather feedback

### Sprint Retrospective
- What went well?
- What can improve?
- Action items

## User Stories

```
As a [role]
I want [feature]
So that [benefit]

Example:
As a gym member
I want to view my workout history
So that I can track my progress
```

## Estimation (Story Points)

- 1: Trivial (< 1 hour)
- 2: Small (1-2 hours)
- 3: Medium (half day)
- 5: Large (1 day)
- 8: Very large (2-3 days)

## Best Practices

✅ Keep sprints consistent  
✅ Prioritize backlog regularly  
✅ Limit WIP (Work In Progress)  
✅ Celebrate wins  
✅ Continuous improvement
