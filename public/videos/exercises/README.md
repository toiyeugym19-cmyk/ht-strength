# Video hướng dẫn bài tập (local)

Đặt file **MP4** hoặc **WebM** vào thư mục này. App sẽ phát trực tiếp không cần API.

## Tên file cần có (đã map trong app)

| File | Bài tập |
|------|--------|
| `bench-press.mp4` | Đẩy ngực ngang (Barbell Bench Press) |
| `deadlift.mp4` | Deadlift |
| `squat.mp4` | Gánh tạ (Squat) |
| `pull-up.mp4` | Hít xà đơn |
| `push-up.mp4` | Hít đất |
| `lat-pulldown.mp4` | Kéo xà cáp (Lat Pulldown) |

## Nguồn video miễn phí (tải về rồi đổi tên cho đúng)

- **Pexels Videos** — https://www.pexels.com/search/videos/gym%20exercise/  
  Tìm "bench press", "squat", "deadlift"... → Download → đổi tên theo bảng trên.
- **Pixabay** — https://pixabay.com/videos/search/workout/  
  Tương tự, tải MP4 rồi đặt vào đây với tên đúng.
- **Videvo** — https://www.videvo.net/free-video/gym/  
  Chọn bản free, tải về, đổi tên.

Sau khi thêm file, reload trang bài tập → mở chi tiết bài → video phát bằng thẻ `<video>` ngay trong web, không cần YouTube.
